# My Portfolio

This is my portfolio built with [NuxtJS](https://https://nuxtjs.org/), [TailwindCSS](https://tailwindcss.com) and [Sanity.io](https://sanity.io)

## Packages I have used :

- nuxtjs/apollo
- graphql-tag
- sanity-blocks-vue-component
- v-hotkey
